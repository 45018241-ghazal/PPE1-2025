Miniprojet

Ce miniprojet nous a servi d’exercice pour nous familiariser avec les outils et les méthodes dont nous aurons besoin pour le projet final. À chaque séance, nous avons enrichi le programme étape par étape.

Comment le miniprojet a été construit

Le projet se construit selon différentes étapes :

  1. Créer un programme qui lit une liste d’adresses, affiche la réponse du serveur, l’encodage et le nombre de mots de chaque page.
  2. Organiser toutes ces données dans un tableau.
  3. Faire en sorte que le programme génère un fichier HTML contenant ce tableau.
  4. Appliquer la mise en forme avec Bulma pour avoir une présentation plus propre.
  5. Publier la page finale sur GitHub.
